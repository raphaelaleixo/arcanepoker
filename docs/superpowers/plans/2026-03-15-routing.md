# Routing Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add React Router v6 with three routes (`/`, `/game`, `/rules`) and a quick-start rules page.

**Architecture:** `BrowserRouter` wraps the app in `main.tsx`. `App.tsx` defines routes using `<Routes>/<Route>`. Three page components live in `src/pages/`. All navigation is contextual (no persistent nav bar). Unknown routes redirect to `/`.

**Tech Stack:** React 18, TypeScript, Vite, MUI v7, `react-router-dom@^6.8.0`, Vitest + @testing-library/react + @testing-library/jest-dom
